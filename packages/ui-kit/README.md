# @tokene/toolkit
These packages aim to provide developers with a set of commonly used components for building TokenE web applications.

![types](https://badgen.net/npm/types/@tokene/ui-kit)

## Getting Started

### Installing

```
yarn add @tokene/ui-kit
```

## Running the tests

```
yarn test
```
