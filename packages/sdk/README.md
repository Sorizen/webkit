# @tokene/sdk
These packages aim to provide developers with a set of commonly used functions and features to interact with TokenE graph and core contracts.

![types](https://badgen.net/npm/types/@tokene/sdk)

## Getting Started

### Installing

```
yarn add @tokene/sdk
```

## Running the tests

```
yarn test
```
